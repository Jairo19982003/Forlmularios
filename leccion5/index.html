Lección 5: Área de Texto (<textarea>)

Para Entradas de Texto Multilínea

Mientras que <input type="text"> es ideal para una sola línea de texto (como nombres, correos, etc.), la etiqueta <textarea> se utiliza cuando necesitas que el usuario pueda introducir texto más largo, como un comentario, un mensaje o una biografía.

A diferencia de <input>, <textarea> es una etiqueta de apertura y cierre. Cualquier texto que coloques entre <textarea> y </textarea> se mostrará como el contenido inicial del campo.

Atributos Principales

  • name: Al igual que con los inputs, es esencial para identificar el dato cuando se envía el formulario.
  • rows: Define la altura visible del área de texto, en número de líneas.
  • cols: Define el ancho visible del área de texto, en número de caracteres.
  • placeholder: Muestra un texto de ejemplo cuando el campo está vacío.
  • required, disabled, readonly: Funcionan igual que en los inputs.

Nota: Aunque rows y cols definen el tamaño inicial, los usuarios generalmente pueden redimensionar el área de texto arrastrando su esquina inferior derecha (este comportamiento se puede controlar con CSS).

Ejemplo de Código


<form action="#">
  <div>
    <label for="comentario">Deja tu comentario:</label>
    <textarea 
        id="comentario" 
        name="user_comment" 
        rows="8" 
        cols="50" 
        placeholder="Escribe aquí tu opinión..."
    ></textarea>
  </div>
  <div>
    <button type="submit">Enviar Comentario</button>
  </div>
</form>

Reto

Crea un formulario de contacto simple. Debe tener un campo para el "Asunto" (usando un <input type="text">) y un campo para el "Mensaje" (usando un <textarea>). El textarea debe tener una altura inicial de 5 filas.
